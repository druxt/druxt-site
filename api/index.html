# DruxtSiteNuxtModule : function

The Nuxt module.

Installs and configures all DruxtJS Site modules.

See: DruxtSiteNuxtModule
Example (nuxt.config.js)

  modules: [
    'druxt-site'
  ],
  druxt: {
    baseUrl: 'https://example.com'
  }
}
