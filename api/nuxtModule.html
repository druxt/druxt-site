# Functions

DruxtSiteNuxtModule(moduleOptions)

Nuxt module function to install Druxt Site.

  • Adds the DruxtSite component.
  • Adds the core modules for DruxtJS Site.
  • Adds default configuration for @nuxtjs/proxy.
  • Enables Vuex store.
